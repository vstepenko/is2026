# API Документація

## Контролер Документів (DocumentController)

Базовий URL: `/api/documents`

### 1. Створити документ
*   **Метод:** `POST`
*   **URL:** `/api/documents`
*   **Опис:** Створює новий документ.
*   **Тіло запиту:** `CreateDocumentRequest`
*   **Відповідь:** `DocumentResponse`

### 2. Оновити документ
*   **Метод:** `PUT`
*   **URL:** `/api/documents/{id}`
*   **Опис:** Оновлює існуючий документ за його ID.
*   **Параметри:** `id` (Long) - Ідентифікатор документа.
*   **Тіло запиту:** `UpdateDocumentRequest`
*   **Відповідь:** `DocumentResponse`

### 3. Видалити документ
*   **Метод:** `DELETE`
*   **URL:** `/api/documents/{id}`
*   **Опис:** Видаляє документ за його ID.
*   **Параметри:** `id` (Long) - Ідентифікатор документа.
*   **Відповідь:** Порожнє тіло (Status 200 OK).

### 4. Отримати документ
*   **Метод:** `GET`
*   **URL:** `/api/documents/{id}`
*   **Опис:** Отримує інформацію про документ за його ID.
*   **Параметри:** `id` (Long) - Ідентифікатор документа.
*   **Відповідь:** `DocumentResponse`
